# 📊 Zepto Sales Dashboard (Power BI)

## 🚀 Overview

This project is an interactive Power BI dashboard built to analyze Zepto product data, including pricing, discounts, stock availability, and ratings.

## 🔧 Tools Used

* Power BI
* Excel / CSV Dataset

## 📈 Features

* KPI cards (Total Sales, Orders, Avg Value)
* Category-based slicer for dynamic filtering
* Visualizations for pricing and discount analysis
* Clean and user-friendly dashboard layout
